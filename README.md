Structure of this repository:

- **code**
  - **sequence_memory.jl:** generative model of sequence memory
  - **test_reconstructions.jl:** tests of reconstruction accuracy on the test sequences
  - **Sequences:** 35 test sequences
  - **Sequence_Lengths:** duplicated versions of the test sequences at varying lengths
- **results**
  - **graphs.ipynb:** notebook for creating graphs
  - **\*.csv:** reconstruction accuracy results
  - **\*.pdf:** graphs
